# Employee Mental Health Treatment Prediction

## Overview
This project predicts whether employees may need mental health support using machine learning.

## Features
- Survey-based data collection
- Machine Learning model (Random Forest)
- HR dashboard with insights
- Privacy-focused (no individual predictions)

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas

## Output
- % of employees needing support
- Work stress analysis
- Key insights & recommendations
